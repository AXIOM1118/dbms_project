# Secure Log Integrity Verification System

## Overview
This project is a DBMS-based system designed to ensure secure and tamper-proof logging using a backend-driven verification mechanism.

## Features
- Log integrity verification
- Secure backend validation
- Dynamic frontend built with React and Vite
- Scalable architecture

## Tech Stack
- Frontend: React.js (Vite)
- Backend: Node.js (planned)
- Database: MongoDB / MySQL
- Version Control: Git and GitHub
